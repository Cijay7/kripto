# Cipher GUI Application

## Documentation
This GUI application provides an intuitive interface to perform various cipher operations, allowing users to encrypt and decrypt messages easily.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Cijay7/kripto.git
   ```
2. Navigate to the project directory:
   ```bash
   cd kripto
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage Guide
1. Run the application:
   ```bash
   python app.py
   ```
2. Use the GUI to input your message, select the desired cipher method, and encrypt or decrypt as needed.
3. Follow the on-screen instructions for operation details.

For further assistance, refer to the official documentation or contact the support team.
